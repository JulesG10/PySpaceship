Bienvue sur SpaceShip

Spaceship est un jeu de combat de vaisseaux en solo ou en multijoueur

Pour pouvoir ajouter vos plugins ou modifier le code rendez-vous dans la partie Fonctionnement, toutes les informations nécessaire vous seront fournie.

Qu'est ce que SpaceShip ?

Le jeu Spaceship est un jeu d’arcade dans lequel on incarne un vaisseau spatial pour combattre des vaisseaux ennemis en multijoueurs ou en solitaire. Dans le mode solo il faut éliminer un maximum d'ennemis tout en survivant le plus longtemps possible pour récolter un grand nombre de points. Le mode multijoueur est un jeu joueur contre joueur qui consiste à détruire un maximum de vaisseaux de adverses contrôlés par d’autres joueurs, afin de marquer un maximum de points pour augmenter la puissance de son vaisseau.

Mode Solo

Grâce au mode solo , affrontez des IA dans un combat spacial sans pitié dans l'atmosphère de l'exoplanète B-13454. Vous y trouverez trois types d'ennemis farouches :

  • Les Cuirassés, de gros vaisseaux lourds avec beaucoup de vie lançants deux missiles puissants d'un coup.
  • Les Croiseurs, des vaisseaux de taille moyenne avec un nombre de points de vie moyen qui tire un ensemble de faibles balles.
  • Les Intercepteurs, qui sont de petits vaisseaux avec peu de vie mais une cadence de tire très élevée et une puissance de frappe aussi lourde que celle des cuirassés.

    • Pour faire face à ces adversaires belliscistes, vous incarnerez le pilote d'un vaisseaux d'une cadence de tire hors-norme ! Mais faites attention à l'état de votre vaisseaux... Votre barre de vie n'est pas infinie et vous devez atteindre votre but: vous devez amasser un maximum de points en éliminants des ennemmis, en survivant longtemps ou encore en rentrant en collision avec un adversaire, ce qui néanmoins peut vous coûter la vie.


    Multijoueurs (beta)

    Avec le mode Multi, combattez vos amis dans une bataille générale aérienne impitoyable ! Vous commencerez par être au commande d'un biplan rapide, mais fébrile... Vous vous renforcerez en abattant vos ennemis et en survivant le plus longtemps possible, de cette manière votre avion gagnera en experience et donc en niveaux aussi bien qu'en vie et en puissance de frappe... mais sa vitesse s'en trouvera fortement diminuée. 12 niveaux sont disponibles, ainsi, en partant d'un biplan, en passant un chasseur et un bombardier, 12 types d'avions sont disponibles et jouables pour vous permettre de devenir le maitre des airs ! Il pourra vous arriver de vous faire éliminer par vos adversaires mais aucune inquiétude ! Le niveau de votre avion n'en sera pas impacté et vous pourrez continuer à règner sur les cieux grâce à vos talent de pilote et la puissance de votre appareil. Pour jouer en multijoueur, vous pouvez héberger ou rejoindre la partie sur un réseau local.

    Fonctionnement de SpaceShip

    Nous allons analyser le fonctionnement interne de code de SpaceShip.

    Moteur

    Le moteur du jeux est entièrement basé sur pygame. Il contient une multitude de fonctions permetant de controller le jeux. Comme par exemple: les graphisme, vecteur, collision, input. Le moteur est également responsable de la gestion des évènements. Il est possible de l'utiliser pour créer des jeux personnalisés.

    TCP Client/Server

    La partie du jeux multijoueur est basé sur le protocole TCP/IP. Il permet de communiquer entre deux joueurs. Le serveur permet de gérer les connexions entrantes et les connexions sortantes pour le jeux. Le client lui communique avec le server pour afficher les autres joueurs en simultaner. Pour pouvoir se connecter le client utilise un code générer en fonction de l'ip du serveur. En voici les fontions: 

    
import string

def encode_ip(ip):
    alphabet = string.ascii_lowercase
    code = ""
    for num in ip.split("."):
        num = int(num)
        if num > 25:
            s = str(num)
            for i in range(0, len(s)):
                code += alphabet[int(s[i])]
            code += "-"
        else:
            code += alphabet[num] + "-"

    return code[:-1]


def decode_ip(code):
    alphabet = string.ascii_lowercase
    ip = ""
    
    for charl in code.split("-"):
        for char in charl:
            ip += str(alphabet.index(char))
        ip += "."

    return ip[:-1]

    Les Paramètres

    Les paramètres du jeux servent a définir les touches du jeux. Par défaut, les touches sont:

  • z Avancer
  • q Gauche
  • d Droite
  • s Bas
  • SPACE ou Click Tirer

    • Plugin & Multi (beta)

    Le joux multijoueur n'est pas encore terminer, mais il est possible de l'utiliser. Pour l'instant, le system de plugin est en cours de développement également. La beta du jeux permet de voir les autres joueurs et les balles qu'ils tirent. Il reste encore à développer les fonctionnalités. Voici une liste des objectifs pour la release final:

  • system de particules
  • plugins
  • collision
  • anti-cheat server system
  • mouvement joueur optimisation
  • sauvegarder les touches dans un fichier
  • générer la map procéduralement
